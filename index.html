The Loop: Ouroboros

The Loop: Ouroboros is a "PT Styled" Indie Horror Game. You need to investigate the mysterious death of a family that used to live in a creepy house in the mountains. By the time you enter the house you realize that something is wrong and something dark is watching you. Every step you take is different and you need to "break" the loop to get free.

Escape The Darkness - GGJ2022

Escape the darkness is a first person horror/thriller game involving a person trying to defeat a monster while trapped in a laminal space with eerie realities. The game follows the main theme of duality in that it is a symbolization of the Abject, a feeling incurred by treading on a plane between two conceptual borders. For this reason the game occurs in an abandoned hospital maze with repetitive rooms/paths that are never quite the same and involve a character who caries a flashlight in a well lit room while chanting Latin prayers to electric switches.

Ritualist

Work In Progress

You are ritualist that needs to finish a ritual in an abandoned graveyard. Getting trapped with an active corpse-demon and escaping alive is not an easy task, but you have enough experience to stay alive in situations like these... Find the parts, collect them, finish the ritual and get away from there. You are not safe until you collect the offering part after the ritual, so be careful!

Hangman - C++ Assignment

This is a C++ assignment that I had to do for CE221 ( C++ Programming). For this assignment we had to create a hangman game. The word was selected randomly from a file. It had 3 difficulties, login system for managers and players, scoring system and tips for the player if they are close to death.

Mars Projection - Java OOP Assignment

This project is an assignment I had to do for CE152 ( Java Object Oriented Programming ) Assignment 2. Create a simple rocket system, crew system, Mars Data plotting/drawing and Solar System animation.

Phobos VR

Brackeys Game Jam 2022

"This is not real"; Our game experiences the theme to its utmost by throwing the player into what is almost a state of delirious struggle for survival. The player is expected to be in the game comparing how much it has in common to reality visually as it behaves in a physically and mechanically surreal way.
Horror; Between reality and fiction.
